704. 二分查找

内容

给定一个n 个元素有序的（升序）整型数组 nums 和一个目标值 target ，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

示例1：

输入: nums = [-1,0,3,5,9,12], target = 9
输出: 4
解释: 9 出现在 nums 中并且下标为 4

示例2：

输入: nums = [-1,0,3,5,9,12], target = 2
输出: -1
解释: 2 不存在 nums 中因此返回 -1

提示：

  1. 你可以假设 nums 中的所有元素是不重复的。
  2. n 将在 [1, 10000]之间。
  3. nums 的每个元素都将在 [-9999, 9999]之间。

思路

在有序数组中进行搜索，一般可考虑使用二分查找。题目明确说明数组中数据有序，因此可以使用二分查找。

这个题是整数二分查找的模版题，直接上模版即可。

二分查找模版思路

我们定义 target 是在封闭区间内的，即 target 在 [ low, high ] 中。因此，有如下特点：

  • while (low <= high) 用 <= ，因为left == right是有意义的，所以使用 <=
  • if (nums[mid] > target) high 要赋值为 mid - 1，因为当前这个nums[mid]一定不是target，那么接下来要查找的左区间结束下标位置就是 mid - 1.
  • If(nums[mid] < target) low 要赋值为 mid + 1， 因为当前这个nums[mid]一定不是target， 那么接下来要查找的右区间开始下标位置就是mid + 1.

代码

class Solution {
    int binarySearch(vector<int> nums, int target)
    {
        int low = 0, high = nums.size()-1;
        while(low <= high)
        {
            int mid = low + (high - low) / 2;
            if(target == nums[mid]) return mid;
            if(target > nums[mid]) low = mid + 1;
            else high = mid - 1;
        }
        return low;
    }
public:
    int search(vector<int>& nums, int target) {
        int id = binarySearch(nums, target);
        if(id >= nums.size()) return -1;
        if(nums[id] == target) return id;
        return -1;
    }
};

34. 在排序数组中查找元素的第一个和最后一个位置

内容

给你一个按照非递减顺序排列的整数数组nums，和一个目标值target。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值target，返回[-1, -1]。

你必须设计并实现时间复杂度为O(log n)的算法解决此问题。

示例 1：

输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]

示例 2：

输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]

示例 3：

输入：nums = [], target = 0
输出：[-1,-1]

提示：

  1. 0 <= nums.length <=10^5
  2. -10^9<= nums[i] <= 10^9`
  3. nums 是一个非递减数组
  4. -109 <= target <= 109

思路

题目中强调整数数组按照非递减排列，因此可以想到使用二分查找算法进行指定元素位置。

由于数组中元素不唯一，因此可以使用改进的二分查找算法来确定给定值在数组中的开始位置；对于结束位置，可以使用同样的查找算法来确定下一个元素在数组中的位置id，则给定值的结束位置即为id-1。

改进的二分查找算法

我们定义 target 是在封闭区间内的，即 target 在 [ low, high ] 中。因此，有如下特点：

  • while (low <= high) 用 <= ，因为left == right是有意义的，所以使用 <=
  • If(nums[mid] < target) low 要赋值为 mid + 1， 因为当前这个nums[mid]一定不是target， 那么接下来要查找的右区间开始下标位置就是mid + 1.
  • If(nums[mid] >= target) high 要赋值为 mid - 1，因为当前这个nums[mid]可能是target，但不一定是 target 序列的开始位置，那么接下来要查找的有取件开始下标位置就是 mid - 1

代码

class Solution {
    int binarySearch(vector<int> nums, int target)
    {
        int low = 0, high = nums.size() - 1;
        while(low <= high)
        {
            int mid = low + (high - low) / 2;
            if(nums[mid] < target) low = mid + 1;
            else high = mid - 1;
        }
        return low;
    }
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        vector<int> ans;
        int id = binarySearch(nums, target);
        if(id >= nums.size() || nums[id] != target){
            ans.push_back(-1);
            ans.push_back(-1);
        }
        else{
            ans.push_back(id);
            id = binarySearch(nums, target+1);
            ans.push_back(id-1);
        }
        return ans;
    }
};

35. 搜索插入位置

内容

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为O(log n) 的算法。

示例 1:

输入: nums = [1,3,5,6], target = 5
输出: 2

示例 2:

输入: nums = [1,3,5,6], target = 2
输出: 1

示例 3:

输入: nums = [1,3,5,6], target = 7
输出: 4

提示:

  1. 1 <= nums.length <= 10^4
  2. -10^4 <= nums[i] <= 10^4
  3. nums为无重复元素的升序排列数组
  4. -10^4 <= target <= 10^4

思路

题目中已知数组排好序，且查找算法时间复杂度需为O(log n)则考虑使用二分查找算法。

该题目为二分查找模板题，直接套用模板即可。

代码

class Solution {
    int binarySearch(vector<int> nums, int target)
    {
        int low = 0, high = nums.size() - 1;
        while(low <= high)
        {
            int mid = low + (high - low) / 2;
            if(nums[mid] == target) return mid;
            if(nums[mid] < target) low = mid + 1;
            else high = mid - 1;
        }
        return low;
    }
public:
    int searchInsert(vector<int>& nums, int target) {
        return binarySearch(nums, target);
    }
};

69.x 的平方根

内容

给你一个非负整数x ，计算并返回x的算术平方根。

由于返回类型是整数，结果只保留整数部分，小数部分将被舍去 。

注意：不允许使用任何内置指数函数和算符，例如pow(x, 0.5)或者x ** 0.5

示例 1：

输入：x = 4
输出：2

示例 2：

输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。

提示：

  1. 0 <= x <= 2^31 - 1

思路

给定非负整数x，求其算术平方根的整数部分，最直接的方式就是暴力穷举，从0开始遍历，遍历上限为x

但暴力穷举时间复杂度过大，因此需要考虑使用高级算法。暴力穷举的思想即为从给定区间中查找目标值，因此可考虑使用二分查找算法，查找区间为[0, x]。

再次阅读题目，不难发现要寻找的值，其平方并不一定等于x，因此考虑查找满足mid^2 <= x + 1的mid-1作为答案。因此直接套用改进的二分查找模板即可。

注意：提示中显示x的范围在运算过程中可能会超出int所表示范围，因此考虑使用long long作为数据类型。

代码

class Solution {
    long long binarySearch(long long x)
    {
        long long low = 0, high = x;
        long long target = x + 1;
        while(low <= high)
        {
            long long mid = low + (high - low) / 2;
            if(mid * mid >= target) high = mid - 1;
            else low = mid + 1;
        }
        return low - 1;
    }
public:
    int mySqrt(int x) {
        return binarySearch(x);
    }
};

367.有效的完全平方数

内容

给定一个正整数num ，编写一个函数，如果num是一个完全平方数，则返回true，否则返回false。

进阶：不要使用任何内置的库函数，如sqrt。

示例 1：

输入：num = 16
输出：true

示例 2：

输入：num = 14
输出：false

提示：

  1. 1 <= num <= 2^31 - 1

思路

给定正整数num，判断是否为完全平方数，实际上就是判断能否在[0, num]区间内找到正整数mid使得mid * mid == num，最直接的方式就是暴力穷举，从0开始遍历，遍历上限为num

但暴力穷举时间复杂度过大，因此需要考虑使用高级算法。暴力穷举的思想即为从给定区间中查找目标值，因此可考虑使用二分查找算法，查找区间为[0, num]。直接套用二分查找模板即可。

注意：提示中显示x的范围在运算过程中可能会超出int所表示范围，因此考虑使用long long作为数据类型。

代码

class Solution {
    bool binarySearch(int num)
    {
        long long low = 0, high = num;
        while(low <= high)
        {
            long long mid = low + (high - low) / 2;
            if(mid * mid == num) return true;
            if(mid * mid < num) low = mid + 1;
            else high = mid - 1;
        }
        return false;
    }
public:
    bool isPerfectSquare(int num) {
        return binarySearch(num);
    }
};
